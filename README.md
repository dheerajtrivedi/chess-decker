# Chess Ԁecker ♟️

A modern chess puzzle trainer inspired by the **Woodpecker Method**, designed to improve pattern recognition through deliberate repetition and performance tracking.

## Features

### 🎯 Standard Puzzle Sets

Create custom puzzle sets by selecting:

* Number of puzzles
* Rating range
* Themes (optional)

For every training session, Chess Ԁecker tracks:

* Accuracy
* Solve time
* Best solve time
* Last solve date
* Historical performance across attempts

This allows you to monitor improvement over time and identify strengths and weaknesses.

---

### 🪵 Woodpecker Method Training

Chess Ԁecker includes dedicated support for the famous **Woodpecker Method**, a training system popularized by Grandmasters Axel Smith and Hans Tikkanen.

Instead of solving new puzzles every day, you repeatedly solve the **same puzzle set** under increasingly strict time constraints.

#### Training Cycles

| Cycle   | Time Limit |
| ------- | ---------- |
| Cycle 1 | 4 Weeks    |
| Cycle 2 | 2 Weeks    |
| Cycle 3 | 1 Week     |
| Cycle 4 | 3 Days     |
| Cycle 5 | 1 Day      |

The goal is not merely to solve puzzles correctly, but to develop automatic pattern recognition through repetition.

### Important

⚠️ **All progress is stored locally.**

If you:

* Clear browser data
* Clear site storage
* Use private/incognito mode
* Switch browsers or devices

your puzzle sets and progress will be permanently lost.

Consider periodically exporting or backing up your data if long-term tracking is important.

## Why Chess Ԁecker?

Most chess trainers focus on solving more puzzles.

Chess Ԁecker focuses on solving the **right puzzles repeatedly** until tactical patterns become instinctive.

Whether you're building custom tactical sets or following a structured Woodpecker cycle, Chess Ԁecker helps transform tactical knowledge into tactical intuition.
